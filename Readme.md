- Initialize git: -`git init`

- See file changes: -`git status`

- See Logs:
  -- `git log`
  -- `git log --oneline`

- Add to Staging area:
  -- `git add <filename>`
  -- `git add .` - add all files of current dir to staging area

- Add to commit:
- `git commit -m "message"`

- Add remote origin:
- `git remote add origin <origin url: https/ssh>`

- Remove origin:
- `git remote remove origin`
